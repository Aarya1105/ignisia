# ignisia

# 🚀 Journey to Ignisia Hackathon – MIT WPU

An interactive storytelling website showcasing our complete journey at **Ignisia Hackathon, MIT World Peace University (MIT WPU), Pune**.

This project presents our hackathon experience through cinematic scrolling animations, smooth transitions, and immersive storytelling.

---

## 🌟 Live Website

👉 https://Aarya1105.github.io/ignisia/

---

## 📖 About the Project

This website narrates our end-to-end hackathon journey:

* ✅ Online Screening Round Presentation
* ✅ Shortlisting among **77 Teams (Rank 18)**
* ✅ Travel from Nashik to Pune
* ✅ Campus Experience at MIT WPU
* ✅ Hackathon Development Journey
* ✅ Evaluations & Twist Rounds
* ✅ Final Results & Certification
* ✅ Memories and Team Experience

The website uses modern web animation libraries to create a cinematic storytelling experience.

---

## 🧑‍💻 Tech Stack

## 🚀 Installation & Run Locally

1. Clone the repository

```
git clone https://github.com/Aarya1105/ignisia.git
```

2. Open folder in VS Code

3. Run using Live Server

```
Right Click → index.html → Open with Live Server
```

---

## 🌍 Deployment

The website is deployed using **GitHub Pages**.

Steps:

* Push code to GitHub
* Go to Repository → Settings → Pages
* Select branch **main**
* Deploy
